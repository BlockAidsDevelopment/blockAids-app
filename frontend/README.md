## Installation

##### Requirements:

- node >=18

##### Tech stack:
-  <a href="https://react.dev/" target="blank">React</a> Frontend Framework

##### Standard Frontend running

- Setup **frontend/.env** file
- Install:

```bash 
   npm install
```

```bash 
  npm run start
```


- Receive Address - http://localhost:7010
